Факти про програмування

NASA використовує Python в проектах, що стосується математичних розрахунків польоту.
Python є важливою частиною Google.Python є повсюдно на YouTube.

Більшість веб сторінок написані з використанням більш ніж однієї мови програмування.
Наприклад, сайт електронної пошти напписаний з допомогою CSS, HTML і JavaScript.

У сучасного персонального комп’ютера в десять разів більше потужності, ніж свого часу було потрібно, щоб запустити і посадити людину на Місяць.Прогрес не стоїть на місці.

Java є другим за популярністю мовою і користується великою популярністю серед розробників. Перша назва Java було «Oak».
Він був змінений на Java відділом маркетингу Sun.

JavaScript не має нічого спільного з Java, хоча мови програмування складаються з одного і того ж слова. Дана схожість пов'язана з суперництвом.JаvаSсrіpt розроблялась близько 8 місяців.

Java і Kotlin можуть бути взаємозамінними. Це означає, що деякі частини коду ви можете писати на Java, а деякі — на Kotlin навіть у межах одного проекту (хоча це може бути не найкраща ідея).
